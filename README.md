## 🛒 Project Name: SunCart

### 🌞 Purpose

**SunCart** is a summer-special e-commerce web application designed to showcase and sell seasonal products such as skincare, sunglasses, and other summer essentials. The platform focuses on a clean UI, smooth user experience, and secure authentication.

---

### 🔗 Live URL

👉 https://suncart-e-com.vercel.app/

---

### ✨ Key Features

* 🔐 **Authentication System**

  * Google Login (OAuth)
  * Email & Password Authentication
  * Built with Better Auth

* 🔒 **Protected Routes (Proxy)**

  * User Profile page is protected
  * Product Details page is protected
  * Unauthorized users are redirected to login page

* 👤 **User Profile Management**

  * Update user information
  * Personalized user experience

* 🛍️ **Product Showcase**

  * Summer-themed product listing
  * Product details page (protected)

* ⚡ **Modern UI/UX**

  * Responsive design (mobile + desktop)
  * Smooth animations using Animate.css

* 🔔 **User Feedback**

  * Toast notifications for actions (login, update, etc.)
